SCREENSHOT INSTRUCTIONS
------------------------

grecord_window.png

1. Start Sound Recorder (Applications -> Multimedia -> Sound Recorder).
2. Ensure that the window has focus. If it does not, click on the window to give it focus.
3. Resize the window to a suitable size.
4. Take a screenshot of the window. 
5. Save the file to PNG.
